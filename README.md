## Description

Time tracker app.
Configure database in `config.ts` (has to be MySQL).
Swagger: `localhost:3000/api`
Unfortunately didn't have time to finish all the tests, but the services unit tests are ready.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start
```

## Test
```bash
$ yarn run test
```
